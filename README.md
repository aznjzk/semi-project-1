## 💻 개발내용  
- 모델1 방식을 사용하여 쇼핑몰 구현  
- 쇼핑몰 내에서 관리자, 회원, 비회원의 기능별 차등 부여  
- 관리자는 2개 등급으로 나누어 등급별 관리 권한 분리  
- 관리자는 상품, 주문상태, 문의, 리뷰관리, 고객리스트조회, 직원관리 가능  
-  상품구매, 리뷰작성, 장바구니 기능, 공지 조회, 정보조회 및 변경 가능  
- 비회원은 상품조회, 리뷰조회, 장바구니, 회원가입만 가능
